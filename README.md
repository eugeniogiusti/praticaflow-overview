# PraticaFlow

**Il gestionale all-in-one per CAF, patronati e agenzie per immigrazione.**  
PraticaFlow semplifica la gestione quotidiana del tuo ufficio: pratiche, clienti, scadenze, documenti, appuntamenti, task, pagamenti e molto altro.  
Tutto in un'unica piattaforma, moderna, intuitiva e pronta all'uso.

---

## ✨ Funzionalità principali

- 📋 Checklist pratiche dinamiche in base al tipo selezionato
- 🧾 Documenti con scadenze, etichette, upload da link
- 📅 Appuntamenti collegati ai clienti e sincronizzabili con Google Calendar
- ✅ Task interni con stato e priorità
- 💰 Gestione pagamenti con bilancio mensile
- 🧠 Supporto AI integrato (ChatGPT) per gli operatori
- 📄 Generazione automatica di documenti PDF con dati precompilati
- 🗃️ Archivio intelligente per pratiche concluse o sospese
- 📊 Dashboard riepilogativa con stato dell’attività
- 🔐 Gestione utenti con permessi granulari
